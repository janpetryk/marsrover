# Mars Rover Kata

## Requirements

- Java 11

## Running application

- `./gradlew run -q --console=plain` on *nix systems
- `gradlew.bat run -q --console=plain` on Windows

## Running tests

- `./gradlew clean check` on *nix systems
- `gradlew.bat clean check` on Windows
